# 🔐 Sistema de Login e Cadastro

Projeto de sistema de autenticação simples desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Cadastro de usuário
- Login com validação de credenciais
- Armazenamento de dados no navegador (localStorage)
- Interface simples e funcional

## 💻 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, incluindo manipulação do DOM, eventos e armazenamento de dados no navegador.

## 📌 Como usar

1. Crie um usuário na seção de cadastro  
2. Faça login com os dados cadastrados  
3. O sistema valida as informações inseridas  

## ⚠️ Observação

Este projeto não utiliza banco de dados real e não deve ser usado em produção. É apenas para fins de estudo.

## 📈 Evoluções futuras

- Integração com API
- Banco de dados real
- Autenticação segura
- Melhorias no design
